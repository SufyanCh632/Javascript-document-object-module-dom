# Javascript-document-object-module-dom

A beginner-friendly project for learning and practicing **JavaScript DOM (Document Object Model)** concepts. This repository contains examples and exercises demonstrating how JavaScript interacts with HTML elements to create dynamic and interactive web pages.

## 📖 About

The **Document Object Model (DOM)** is a programming interface that represents an HTML document as a tree structure. JavaScript can use the DOM to access, modify, add, or remove elements on a web page, making websites interactive.

This repository is designed to help beginners understand DOM manipulation through simple examples and practical code snippets.

## 🚀 Topics Covered

* Selecting HTML elements

  * `getElementById()`
  * `getElementsByClassName()`
  * `getElementsByTagName()`
  * `querySelector()`
  * `querySelectorAll()`

* Modifying Elements

  * Changing text content
  * Updating HTML content
  * Modifying CSS styles
  * Changing attributes

* Event Handling

  * Button clicks
  * Mouse events
  * Keyboard events

* Creating and Removing Elements

  * `createElement()`
  * `appendChild()`
  * `remove()`

* DOM Traversal

  * Parent elements
  * Child elements
  * Sibling elements

## 💻 Example

```javascript
document.getElementById("btn").textContent = "Clicked!";
```

This code selects an element with the ID `btn` and changes its text content.
